AI Wine Searcher

Python, React

• Developed a full stack application that finds affordable, “good” wine based on a user’s preferred flavor profile. I utilize preferences, price range, and various wine ranking systems to web scrape wine-searcher.com and vivino.com to pair the user with wines they will love.

Something New:

This project was the first time I integrated a personal passion into a project. I love wine, yet every time I went to a grocery store I got overwhelmed with the selection size and didn’t know which to buy. I grew tired of researching every intriguing label from unfamiliar wines. Maze Solver

C++

• Wrote a depth-first algorithm that randomly generates and solves a maze of arbitrary size. I animate the process through a GUI that allows users to choose the width and height of the maze.


A Humbling Experience:

This was my favorite project in C++ and the project that helped my mastery of the language. The language was unforgiving and I spent more time fixing memory leaks than writing the logic for the maze solver/generator. Prior to this project, I felt confident that I was “good” at writing code because once I figured out how to approach it, nothing could stop me. Yet C++ is the only language that made me pause and question whether or not I was correctly solving a problem. It taught me to slow down to consider all of the complex moving parts.

Sudoku

MIPS Assembly

• Coded a fully customizable Sudoku game that validates user moves, gives hints, and exports board solutions. Has an AI feature that uses first-depth search algorithm to solve any imported Sudoku board.
• Utilized register conventions, epilogues/prologues, and stack pointers to write readable, efficient assembly code.


Leveling Up:

This was my first large project in an Assembly language. Working in a low-level language was a difficult and rewarding experience. MIPS forced me to slow down and think not just about game mechanics but also important backend details such as memory management. I had to properly manipulate bits and manage registrars and memory bocks in an organized manner. This project taught me the importance of comments and code documentation. My struggles from this project gave me a greater appreciation for higher-level languages.
